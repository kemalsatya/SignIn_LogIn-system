# Project Account Log 

Project ini merupakan sistem login dan sign up. Project ini dibuat sebagai bahan latihan sekaligus praktik dari pemahaman singkat terkait frontend dan backend.

## Cara Penggunaan 
1. Instalasi nodejs
2. Instalasi XAMPP atau program sejenis dan import sql dari folder database
3. CLone repository project ini
4. Inisialisasi project ini dengan command ini
```
npm install
```
5. Jalankan project ini dengan command dibawah
```
npm start
```
